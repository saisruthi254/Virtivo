Virtivo – A Collaborative Learning Platform

Project Description

Virtivo is a platform where users can connect and exchange skills. It allows users to learn from others and also teach their own skills.

 Features
>	User Authentication (Signup/Login)
> Skill sharing and exchange system
>	User profile management
> Study Rooms(User can create or join rooms for real-time collaboration)
>	Peer-to-peer learning

 Tech Stack
>	Frontend: HTML, CSS, JavaScript
>	Backend: Supabase
>	Database: PostgreSQL

Database
This project uses Supabase (PostgreSQL).
Refer to database.sql for table structure.

Setup Instructions
1.	Download or clone the repository
2.	Open the project folder
3.	Supabase URL and API Key are already configured for demonstration purposes.
4.	Open landing.html in browser

Project Details

This project solves the problem of accessing affordable learning by allowing users to exchange skills instead of paying for courses.
